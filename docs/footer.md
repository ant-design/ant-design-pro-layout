## 相关 API 展示

### ProLayout

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| footerRender | 自定义页脚的 render 方法 | (props: BasicLayoutProps) => ReactNode | - |

### DefaultFooter

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| links | 默认自带的一些 | false \| `{key:string,title:string,href:string}[]` | - |
| copyright | 版权声明文字 | ReactNode | - |
